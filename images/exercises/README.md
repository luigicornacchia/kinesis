# Immagini Esercizi

Questa cartella contiene le immagini degli esercizi disponibili nell'app.

## Come aggiungere nuove immagini

1. **Aggiungi l'immagine** in questa cartella (`images/exercises/`)
   - Usa nomi file in minuscolo con trattini
   - Formato consigliato: JPG o PNG
   - Esempio: `panca-piana.jpg`, `squat.jpg`

2. **Aggiorna il file** `exercise-images.js`
   - Aggiungi una nuova riga nell'array `exerciseImages`
   - Esempio:
   ```javascript
   { name: 'Panca Inclinata', path: 'images/exercises/panca-inclinata.jpg' }
   ```

3. **Fai commit e push** su GitHub

## Immagini da aggiungere (esempi)

Per iniziare, puoi scaricare immagini gratuite da siti come:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)

Cerca per: "gym exercise", "workout", "fitness"

## Note

- Le immagini verranno ridimensionate automaticamente dall'app
- Dimensione consigliata: max 800x800px per bilanciare qualità e velocità
- Assicurati di avere i diritti per usare le immagini
